Привіт, це программа створена лише на пайтон використовуючи фреймворк кастомткінтер.
 Це невелечкий симулятор пральноЇї машинки де вам потрібно вибирати тип прання ласьони та порошки й авжеж речі які будуть пратись.
 Ви побачите результат вашого прання виведений на екран, він буде залежати від вибору компонентів.
 Але не забудьте вибрати всі компоненти! Бо пральна машинка не сможе працювати без них.
 Тож гарного вам часопроводження!

 Hello, this program is created only in python using the customkinter framework.
 This is a small simulator of a washing machine, where you need to choose the type of laundry detergent and powder and, of course, the things that will be seen.
 You will see the result of your washing displayed on the screen, it will depend on the selection of components.
 But don't forget to select all components! Because the washing machine will not be able to work without them.
 So have a good time!
